# Server

Servidor escrito en Golang para el juego "El pibe gravedad".

## Ejecucion local

La ejecucion local del proyecto es utilizando `docker` y `make`:

1. Instala docker y el plugin compose
2. Instala make
3. Ejecuta

    ```bash
    make dev
    ```

O directamente usando `go`:

1. Instala go>1.23.1
2. Ejecuta

    ```bash
    go run .
    ```
